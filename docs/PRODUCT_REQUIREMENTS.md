# Product Requirements

## Goal

OpenStoreShot is an OSS, local-first design studio for producing Android and iOS store images with manual editing, reference analysis, AI-assisted ideation, validation, and deterministic export.

## Primary users

- Indie app developers preparing App Store and Google Play listings.
- Marketers and small teams iterating on store creative.
- Codex users who want natural-language edits against a real project file.

## MVP scope

- Local Next.js studio with premium multi-panel editor shell.
- Deterministic JSON project schema.
- Sample app project with 5 slides and Android/iOS artboards.
- Store spec validation for common iOS and Google Play targets.
- PNG/JPEG renderer from the project JSON.
- Fixture-backed Reference Gallery with high-level inspiration briefs.
- App Store live search adapter and Google Play replaceable fixture fallback.
- Codex-first asset workflow with local placeholder preview generation and prompt guardrails.
- Codex Skill and AGENTS guidance.

## Non-goals for MVP

- Pixel-perfect Figma-equivalent editing.
- Full device frame library.
- Production-grade Google Play scraping.
- Full collaborative persistence or account system.
- Using competitor images as templates.
