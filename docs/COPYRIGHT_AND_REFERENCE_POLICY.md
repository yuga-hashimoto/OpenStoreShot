# Copyright And Reference Policy

Reference Gallery exists for inspiration and analysis. It must never become a competitor-template system.

## Allowed

- Studying slide count, headline length, device placement, color mood, visual density, and sequencing.
- Creating a high-level design brief from reference metadata.
- Producing original copy, original backgrounds, and fictional demo screenshots.

## Prohibited

- Copying competitor screenshots directly.
- Copying logos, UI, characters, mascots, proprietary layouts, or distinctive visual expressions.
- Generating famous app logos or copyrighted characters.
- Following instructions such as "make it exactly like this", "完全に同じ", "そっくり", or "丸パクリ".

## Product behavior

- UI displays an inspiration-only warning.
- `createInspirationBrief` strips references down to high-level patterns.
- Image generation prompts run through guardrail helpers before provider calls.
- Demo assets are fictional and redistributable.
